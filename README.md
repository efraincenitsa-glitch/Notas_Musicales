
# Piano IA Detector

Aplicación web que:

- Detecta notas musicales en tiempo real usando el micrófono
- Identifica acordes automáticamente con Tonal.js
- Muestra un piano de 88 teclas
- Guarda una lista de notas detectadas
- Se puede instalar en iPhone como Web App

## Cómo usar

1. Subir estos archivos a un repositorio de GitHub
2. Activar GitHub Pages
3. Abrir la página en el navegador
4. Presionar **Activar micrófono**

## Librerías

- Tonal.js (detección avanzada de acordes)
- Web Audio API
